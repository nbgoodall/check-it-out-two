# check-it-out-two

A simple animation app built in react.

## Contributers

- mrbenbot
- nbgoodall

## Features

- [x] Export as gif
- [x] Undo
- [x] Change frame rate
- [ ] Edit an existing frame
- [ ] Fork existing animation
- [ ] Save
- [ ] Delete frame
- [ ] copy/paste area of a frame
- [ ] fill area of a frame
